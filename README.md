# OSSTEM E9

오스템임플란트 해외관리2팀 사내 게임 앱입니다.

## 게임 모드

### 뽑기
- 여러 개의 동심원 링 안에 참가자 공이 갇혀 있습니다.
- 링마다 열린 구역(출구)을 통해 공이 안쪽으로 이동하며, 최종적으로 중심을 탈출하는 공이 당첨됩니다.

### 사이드 메뉴 쟁탈전
- 모든 참가자가 탈출할 때까지 게임이 진행됩니다.
- 누군가 탈출할 때마다 남은 공의 속도가 빨라집니다.
- 탈출 순서대로 최종 순위가 결정됩니다.

## 실행 방법

`lottery.html` 파일을 브라우저에서 열거나, 배포된 링크로 접속하세요.

## 기술 스택

- HTML / CSS / JavaScript (단일 파일)
- Canvas 2D API (물리 엔진)
- Web Audio API (BGM)
